This project consists of three exercises as an introduction to C++.
It's a part of 42 Network Core Curriculum CPP exercises.

This module is focused mostly on fixed numbers. It also teaches the basics of ad-hoc polymorphism, operator overloading
and the Orthodox Canonical class form.

The first exercise is about learning about fixed numbers and the orthodox canonical form. The second exercise is about convertion and better understanding of fixed numbers. The third one is mostly about the operator overloading.
