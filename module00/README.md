
This project consists of two short exercises as an introduction to C++.
It's a part of 42 Network Core Curriculum CPP exercises.

It covers basics as namespaces, classes, member functions, stdio streams, initialization lists, static, const and other basic stuff.

First exercise mimic the behaviour of megaphone and convert all input strings into uppercase.

Second exercise should behave as a simplified phonebook, being able to add contacts, search thourg them and exit.
