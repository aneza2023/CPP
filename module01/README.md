This project consists of five mandatory short exercises and one non-mandatory as an introduction to C++.
It's a part of 42 Network Core Curriculum CPP exercises.

It covers basics od memory allocation, pointers to members, references and switch statements

The first exercise is focused on difference between allocation on the heap or the stack and proper deletion. The second is about allocating N numbers of obejcts in a single allocation. Third one is about the difference between pointers and references. The fourth is about he difference of using pointer or a reference within a class. The fifth exercise shows basics of working with files in C++ and the last mandatory exercise is about using pointers to member functions. 
Final last exercise is about switch statements.
